## Aim of project

The broad aim of the project is to provide a full computerised cancellation task,
with a comprehensive and automated analysis.

The project will be user friendly even for non-technical users as well as being,
high-performance and accurate. 

## Psychtoolbox version

The matlab version of this project was built with psychtoolbox version: 3.0.19.15 

Note: The matlab version of this project will be later replaced as the main version,
for the experimental software. However the matlab version will still be avaliable,
but only updated with critial bug fixes.
