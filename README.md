## Aim of project

The broad aim of the project is to provide a full computerised cancellation task,
with a comprehensive and automated analysis.

The project will be user friendly even for non-technical users as well as being,
high-performance and accurate. 

## Future plans

 - BIDS compatiable.
 - Python version of experiment program.
 - State of the art spatial attentional analysis.
 - Extensive documentation on how to use program.
 - Eye-Tracking support.
 - Neuroimaging support (OPM-MEG/SQUID MEG, EEG, MRI).
 - High-Performance Rust Core.
